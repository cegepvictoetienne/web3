# Inscription à Azure  


Microsoft offre à tous les élèves un crédit annuel de 100 $US. Il suffit de s'inscrire au produit __Azure for Students__ avec votre courriel du Cégep (Outlook), à l'adresse suivante [https://azure.microsoft.com/fr-ca/free/students?icid=portal](https://azure.microsoft.com/fr-ca/free/students?icid=portal)
